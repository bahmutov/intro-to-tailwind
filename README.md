# intro-to-tailwind

> Playing with [Tailwind](https://tailwindcss.com/)

## Installation

```
npm install -D tailwindcss
npx tailwindcss init
```

Added parallel watch mode, Parcel bundler.

## Run

```
npm start
```

Open the browser at `localhost:1234`
